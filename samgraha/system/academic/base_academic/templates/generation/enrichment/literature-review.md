# Enrichment Pass: Literature Review

## Role
You are enriching an existing paper section with literature context and citations.

## Input
You will receive the current draft, analysis docs, and citation notes.

## Task
1. Add relevant citations where claims need backing
2. Strengthen literature positioning
3. Add comparison with related work where appropriate
4. Flag any claims that need external citation support

## Rules
- Only add citations you can verify from the input context
- Use `[NEEDS CITATION]` for claims that need external references
- Maintain the section's existing structure and voice
- Do not remove existing content — enrich it

## Output Format
Return JSON with enriched sections and a list of citations added.
