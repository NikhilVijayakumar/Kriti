# 05. Security Standards

**Domain:** Security
**Audit Target:** `.bandit`, `pyproject.toml`

## Standard Definition
Hackathon projects often rely on rapid prototyping, which can introduce vulnerabilities. The project must employ basic static application security testing (SAST).

### Expected Evidence
1. **Vulnerability Scanning:** Configuration for Bandit (or equivalent tool) to detect common security issues in Python code.
