# 03. Testing Standards

**Domain:** Testing
**Audit Target:** `tests/` directory, `pytest.ini`

## Standard Definition
Projects must include automated validation to verify core functionality before runtime execution.

### Expected Evidence
1. **Test Suite:** A dedicated `tests/` or `test/` directory must exist.
2. **Runner Configuration:** A `pytest.ini` or equivalent `pyproject.toml` block must define the test execution strategy.
