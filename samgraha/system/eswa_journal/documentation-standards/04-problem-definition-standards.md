## Assumptions
**Template:**
```markdown

1. [Assumption 1]

```

## Writing Guidance
- **Do:** Use consistent notation, define all variables immediately after introducing them.
- **Don't:** Rely solely on programming logic or prose; it must have a formal mathematical/logical foundation.
