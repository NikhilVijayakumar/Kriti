## Purpose
Position the contribution academically by evaluating 15+ recent references (2022-2026), including 2-4 recent ESWA citations.

## Taxonomy Sections
**Template:**
```markdown
[Categorize prior work by approach, e.g., Rule-based systems, Learning-based systems, Hybrid models. Do NOT list chronologically.]
```

## Summary Comparison Table
**Template:**
```markdown
| Study | Approach | Dataset | Limitation |
|---|---|---|---|
| [Citation] | Method used | Data used | The specific gap this paper addresses |
```
**Audit Rule:** A markdown table comparing prior works and explicitly listing their limitations is mandatory.

## Gap-Closing Bridge
**Template:**
```markdown
None of the above approaches adequately address [Specific Gap], which motivates the framework proposed in this paper.
```
**Audit Rule:** The section MUST conclude with a sentence explicitly identifying the unresolved challenge that this paper solves.

## Writing Guidance
- **Do:** Synthesize trends, highlight unresolved challenges.
- **Don't:** Summarize chronologically, cite blogs or non-peer-reviewed URLs.
