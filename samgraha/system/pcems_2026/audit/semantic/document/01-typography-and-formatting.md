# Typography and Formatting Audit

This section details the Typography and Formatting Audit for PCEMS 2026.

## Version
1.0.0

## Engineering Intent
Enforces the strict Arial-based font hierarchy and layout requirements defined in the PCEMS guidelines to prevent desk rejection.

## Audit Objectives
- Verify that the document uses a single-column layout.
- Enforce the precise Heading and Body text font specifications.

## Expected Quality
- Heading 1: Arial, 12 pt bold.
- Heading 2: Arial, 12 pt.
- Heading 3: Arial, 12 pt italics.
- Body text: Arial 11 pt.
- References: 8 pt font.

## Scoring Criteria

| ID | Weight | Score | Description |
|---|---|---|---|
| C1 | mandatory | 0 or 50 | Document uses a single-column layout |
| C2 | mandatory | 0 or 50 | All font families and sizes perfectly match the Arial hierarchy (Heading 1/2/3, Body, References) |
